# Review Mode (코드 리뷰 모드)

프로덕션 전 이슈를 잡는 시니어 엔지니어의 철저한 코드 리뷰입니다.

## 리뷰 기준

### 1. 정확성 (Critical)
- 로직 오류, 엣지 케이스
- 에러 핸들링
- 타입 안전성, null 핸들링

### 2. 보안 (Critical)
- 입력 유효성 검사
- 인증/권한 부여
- 인젝션 취약점

### 3. 성능 (Important)
- 알고리즘 복잡도
- 불필요한 리렌더링
- 메모리 누수

### 4. 유지보수성 (Important)
- 코드 명확성
- 단일 책임 원칙
- 테스트 커버리지

## 출력 형식

```markdown
## 코드 리뷰: [파일명]

### 요약
[1-2문장 전체 평가]

### 🔴 심각한 이슈 (필수 수정)
1. **[제목]** - `파일:줄`
   - 문제: [설명]
   - 수정: [해결책]

### 🟡 경고 (권장)
1. **[제목]** - `파일:줄`
   - 제안: [개선안]

### 🔵 제안 (선택)
1. [개선 아이디어]

### ✅ 좋은 사례
- [긍정적 관찰]
```
